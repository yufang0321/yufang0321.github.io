邏輯空賦值

By 芳芳 09 August 2024

工作中遇到後端傳回參數可能有null及空字串的狀況，必須判斷是否為null或空字串，如果是再將預設的值塞入

故認真的來了解一下false family及如何判斷是否有值及賦值

這篇是把相關閱讀到的概念都整理成一篇

falsy Family

let a = undefined;
let b = null;
let c = '';

console.log(!!a); //false
console.log(!!b); //false
console.log(!!c); //false

JavaScript的強制轉型

const a = "123"
const b = 100
const c = undefined
const d = null
const e = ''
    
if(!!a){ //true
    console.log(a)
}
if(!!b){ //true
    console.log(b)
}
if(!!c){ //false
    console.log(c)
}
if(!!d){ //false
    console.log(d)
}
if(!!e){ //false
    console.log(e)
}

查找資料時發現這個，覺得整理得很棒，整理過來參考

value !value !!value
false true false
true false true

空值合併運算符 ??

const a = null ?? 'default string';
console.log(a); //"default string"

const b = 0 ?? 50;
console.log(b); //0

邏輯空賦值運算符 ??=

const a = { duration: 50 };

a.duration ??= 10;
console.log(a.duration); //50

a.speed ??= 25;
console.log(a.speed); //25

三元運算子

function getFee(isVIP) {
  return isVIP ? 'FREE' : '$10.00';
}

console.log(getFee(true)); //'FREE'
console.log(getFee(false)); //"$10.00"
console.log(getFee(null)); //"$10.00"
